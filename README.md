# React Router Example

How to make a Single Page Application with React Router.

### [View Tutorial](https://www.taniarascia.com/using-react-router-spa)

## Purpose

Set up a router in React.

- `/` - root
- `/:page_id` - page
- `/categories/:category_id` - category
